# Nirvana Labs Examples

Ready-to-deploy examples for [Nirvana Labs](https://nirvanalabs.io) cloud infrastructure. Each example includes Terraform for provisioning and Ansible for configuration.

## Getting Started

1. [Create an account](https://nirvanalabs.io/sign-up) on Nirvana Labs
2. Create an API key from the dashboard
3. Pick an example in from [here](https://github.com/nirvana-labs-examples)
